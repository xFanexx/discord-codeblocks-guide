# 💡 Code Examples for Supported Languages

This document provides minimal code examples for each language listed in `languages.md`.
Use these snippets to test syntax highlighting or copy as templates.

---

## 🐍 Python (`python`, `py`)

```python
def greet(name):
    print(f"Hello, {name}!")

greet("World")
```

